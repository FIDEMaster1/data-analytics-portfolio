## Salary Prediction (NLP + DL)

Проект предсказывает зарплату по тексту вакансии и категориальным признакам. Данные очищены и токенизированы, построен словарь с обработкой редких слов, текст переведен в числовые последовательности, категориальные признаки закодированы через one-hot. Целевая переменная логарифмирована для стабильности обучения. Обучена нейросетевая модель на PyTorch: эмбеддинги + CNN для `Title` и `FullDescription`, объединение с категориальными фичами и регрессия. Реализован пайплайн батчей, обучение и валидация, а также простая интерпретация модели через анализ вклада слов в предсказание.

**Технологии:** NumPy, Pandas, NLTK, scikit-learn, PyTorch, Matplotlib
