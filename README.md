# 📊 Data Analytics Portfolio

Набор pet-проектов по аналитике данных, продуктовой аналитике и машинному обучению. Включает полный цикл работы с данными: от сбора и EDA до моделирования и визуализации.

---

## 📂 Проекты

### 🛒 E-Commerce EDA (Olist)
- Исследовательский анализ e-commerce датасета: поведение пользователей, логистика, доставка и её влияние на оценки, рост заказов и выручки.
- **Технологии:** pandas, numpy, matplotlib, seaborn

### 📊 E-commerce Product Analytics
- Продуктовый анализ пользовательского поведения: воронка, когорты, retention, LTV, анализ каналов и устройств, выявление узких мест.
- **Технологии:** SQL (BigQuery), pandas, matplotlib, seaborn

### 📊 Online Retail Dashboard
- Интерактивный BI-дашборд: выручка, заказы, клиенты, география и динамика продаж.
- **Технологии:** Power BI, DAX, Power Query

### 🏦 CBR Monetary Policy Scraper
- Парсинг и подготовка корпуса текстов Банка России: сбор, очистка HTML, нормализация и сохранение структурированных данных.
- **Технологии:** requests, playwright, BeautifulSoup, regex, JSONL

### 🧪 A/B Testing
- Полный цикл A/B тестирования на e-commerce данных: подготовка user-level датасета, формулировка гипотез, симуляция эксперимента, проверка через bootstrap и z-test, применение CUPED.
- **Технологии:** pandas, numpy, matplotlib, seaborn, статистика

### 🎵 Spotify Songs Popularity Prediction
- Реализация Random Forest с нуля: собственный Decision Tree, bootstrap, OOB-оценка и подбор гиперпараметров.
- **Технологии:** numpy, pandas

### 🌪 NLP Disaster Tweets Classification
- Классификация твитов на реальные катастрофы с полным NLP-пайплайном: preprocessing, векторизация, обучение и сравнение моделей.
- **Технологии:** pandas, numpy, scikit-learn, nltk

### 💼 Salary Prediction
- Предсказание зарплаты по тексту вакансии с использованием нейросети: обработка текста, embedding, CNN и объединение с категориальными признаками.
- **Технологии:** PyTorch, pandas, numpy, nltk, scikit-learn
