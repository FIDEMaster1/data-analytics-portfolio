# Olist E-Commerce EDA

EDA проекта на датасете **Brazilian E-Commerce Public Dataset by Olist**.

## Ключевые выводы

- Почти **98% заказов доставлены**.
- Заказы и выручка росли по мере развития платформы.
- В будние дни заказов больше, чем в выходные.
- Большинство заказов — с одним товаром и небольшим чеком.
- Стоимость доставки сильнее связана с весом и объёмом товара, чем с ценой.
- Большинство заказов доставляется раньше обещанного срока.
- Задержка доставки резко снижает `review_score`.
- Основной рынок платформы — штат **São Paulo**.
- Долгая доставка чаще встречается у крупных товаров: мебель, интерьер, бытовая техника.

## Технологии

- `pandas` — загрузка, merge таблиц, агрегации, обработка пропусков
- `numpy` — работа с числовыми признаками
- `matplotlib` — базовая визуализация
- `seaborn` — графики распределений, barplot, scatterplot, heatmap
- `datetime features` — работа с датами и временными признаками
- `feature engineering` — создание логистических, товарных и платежных признаков
