# Online Retail Dashboard

Дашборд построен в Power BI на основе датасета Online Retail и отражает ключевые метрики интернет-магазина: выручку, количество заказов и клиентов, динамику продаж по месяцам, распределение выручки по странам и товарам, а также географию продаж. Выполнена базовая очистка и трансформация данных в Power Query. Дашборд позволяет быстро оценить тренды продаж, выявить ключевые рынки и наиболее прибыльные товары, а также анализировать изменения показателей во времени с помощью интерактивного фильтра по дате.

**Инструменты:** Power BI, Power Query, DAX  
**Файлы:** `.pbix`, `.pdf`, `.xlsx`
