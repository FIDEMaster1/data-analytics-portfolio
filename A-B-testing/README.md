# A/B тестирование на данных e-commerce

В проекте реализован полный цикл A/B тестирования на данных интернет-магазина: проведён *EDA*, из event-level данных собран *user-level датасет*, сформулирована гипотеза и дизайн эксперимента, выполнена симуляция A/B теста с реалистичным вмешиванием эффекта. Определены и обоснованы метрики (*primary — revenue per user*, *proxy — конверсии и поведенческие метрики*), проведена проверка гипотез с использованием *bootstrap* для скошенных распределений выручки и *z-test* для бинарных метрик. Дополнительно применён метод *CUPED* для снижения дисперсии и повышения чувствительности эксперимента.

## ⚙️ Технологии
* Python (pandas, numpy)
* matplotlib, seaborn
* Статистика: bootstrap, hypothesis testing, z-test, CUPED
